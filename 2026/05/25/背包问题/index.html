背包问题

2026/5/25 算法模板 c++ 算法模板

1. 01背包

  • 特点：每种物品最多选一次。
  • 状态：dp[j] 表示容量 j 时的最大价值。
  • 转移：dp[j] = max(dp[j], dp[j - w[i]] + v[i])
  • 实现：一维数组，容量倒序循环。
for (int i = 1; i <= n; ++i)
    for (int j = V; j >= w[i]; --j)
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);

2. 完全背包

  • 特点：每种物品无限次选择。
  • 状态：同上。
  • 转移：dp[j] = max(dp[j], dp[j - w[i]] + v[i])
  • 实现：一维数组，容量正序循环（允许同一物品多次叠加）。
for (int i = 1; i <= n; ++i)
    for (int j = w[i]; j <= V; ++j)
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);

3. 多重背包

  • 特点：每种物品有数量限制 c[i]。
  • 优化方法：
    • 二进制拆分：将 c[i] 拆成 1,2,4,...,k 若干组，每组视为一个01背包物品，复杂度 O(V Σ log c[i])。
    • 单调队列优化：按余数分类，用单调队列维护滑动窗口最大值，复杂度 O(nV)。

二进制拆分核心代码

for (int i = 1; i <= n; ++i) {
    int k = 1, rest = c[i];
    while (k <= rest) {
        int weight = k * w[i], value = k * v[i];
        for (int j = V; j >= weight; --j)
            dp[j] = max(dp[j], dp[j - weight] + value);
        rest -= k;
        k <<= 1;
    }
    if (rest) { /* 剩余部分同样处理 */ }
}

4. 分组背包

  • 特点：物品分为若干组，每组内最多选一个。
  • 状态：dp[j] 表示容量 j 的最大价值。
  • 转移：先遍历组，再遍历容量（倒序），最后遍历组内物品。
for (int k = 1; k <= groups; ++k)
    for (int j = V; j >= 0; --j)
        for (auto &[w, v] : items_in_group[k])
            if (j >= w) dp[j] = max(dp[j], dp[j - w] + v);

5. 依赖背包（树形背包）

  • 特点：物品有依赖关系（选 u 必须先选 v），通常以树形结构给出。
  • 状态：dp[u][j] 表示以 u 为根的子树中，容量 j 的最大价值。
  • 转移：对子节点进行分组背包（先递归子节点，再合并）。常见于“有依赖的背包”问题，也可转化为“森林+虚拟根”。
void dfs(int u) {
    // 初始化：选 u 本身
    for (int j = w[u]; j <= V; ++j) dp[u][j] = v[u];
    for (int child : tree[u]) {
        dfs(child);
        // 合并子节点：j从V downto w[u]
        for (int j = V; j >= w[u]; --j)
            for (int k = 0; k <= j - w[u]; ++k)
                dp[u][j] = max(dp[u][j], dp[u][j - k] + dp[child][k]);
    }
}

6. 二维费用背包

  • 特点：每件物品消耗两种资源（容量1、容量2），每种资源有限制。
  • 状态：dp[j][k] 表示使用容量1为 j、容量2为 k 时的最大价值。
  • 转移：根据物品类型（01/完全/多重）选择合适的遍历顺序（一般是两维倒序）。
// 01背包版本
for (int i = 1; i <= n; ++i)
    for (int j = V1; j >= w1[i]; --j)
        for (int k = V2; k >= w2[i]; --k)
            dp[j][k] = max(dp[j][k], dp[j - w1[i]][k - w2[i]] + v[i]);

7. 混合背包

  • 特点：同时存在01、完全、多重背包的物品。
  • 处理方法：分别处理，01和多重用倒序，完全用正序。

8. 背包问题的常见变种

(1) 恰好装满

  • 初始化：dp[0] = 0，其他 dp[j] = -INF。
  • 最终：若 dp[V] < 0 则表示无法恰好装满。

(2) 求方案数（最大价值下的方案数或装满的方案数）

  • 状态：dp[j] 表示最大价值，cnt[j] 表示达到该价值的方案数。
  • 转移时：若新价值 > 旧价值，则 cnt[j] = cnt[j - w]；若相等，则累加。
  • 初始化：dp[0] = 0, cnt[0] = 1。

(3) 输出具体方案（任意一种或字典序最小）

  • 方法：记录 choice[i][j] 表示状态 (i,j) 是否选取了物品 i。最后倒推。
  • 字典序最小：通常将物品编号倒序处理，或从后向前DP，再正序贪心。

(4) 求最优方案（价值最大，其次物品数量最少/最多等）

  • 状态：使用结构体或两维状态 dp[j] = {maxValue, minCount}，自定义比较规则。
